## Privacy Policy

**Last updated:** 2026

NamaStream does not collect, store, or transmit any personal data.

- **No accounts.** No login, no credentials stored.
- **No tracking.** No analytics or telemetry of any kind.
- **Stream data.** The Cloudflare Worker fetches public YouTube and Twitch data on behalf of the browser. No user-identifiable information is sent to the Worker.
- **YouTube API.** This extension uses YouTube API Services. YouTube may collect data as described in the [Google Privacy Policy](https://policies.google.com/privacy).
- **Twitch API.** Twitch may collect data as described in the [Twitch Privacy Policy](https://www.twitch.tv/p/legal/privacy-policy/).
- **Location data.** The weather widget optionally requests your approximate location via the browser Geolocation API. This data is sent directly from your browser to [Open-Meteo](https://open-meteo.com/en/terms) (weather forecasts) and [Nominatim / OpenStreetMap](https://osmfoundation.org/wiki/Privacy_Policy) (reverse geocoding). NamaStream does not receive, store, or log your location. Location is only requested if you interact with the weather widget.
- **Local storage.** All user preferences (wallpapers, pinned channels, bar positions, layout settings) are stored locally via the Web Extensions storage API and never leave your device.
    
For questions, open an issue in this repository.
